This file merges the content of netbox_crud_ops.py + netbox_query_ops.py + the delegation logic from the NetBoxStorage class. It exposes module-level async functions (no class wrapper). Functions to implement:

```python
async def get_segments(
    site: Optional[str] = None,
    vrf: Optional[str] = None,
    vlan_id: Optional[int] = None,
    allocated: Optional[bool] = None,
    cluster_name: Optional[str] = None,
    released: Optional[bool] = None,
) -> List[Dict[str, Any]]:
```
- Logic: get RedBull tenant_id via `_objects.get_redbull_tenant_id()`, build `nb_filter = {"tenant_id": tenant_id}`, add `vlan_vid` if vlan_id given.
- Cache key = CACHE_KEY_PREFIXES. Check cache, handle inflight request coalescing (same pattern as current netbox_query_ops.py find()).
- Fetch via `run_netbox_get(lambda: list(nb.ipam.prefixes.filter(**nb_filter)), "fetch prefixes")`.
- Convert each prefix: `segment = prefix_to_segment(prefix)` (1-arg — nb_client param is gone).
- Skip segments with missing site or vrf.
- Apply Python-native filters using `_segment_matches()` private helper:
  - `site`: `segment["site"].lower() == site.lower()` (case-insensitive)
  - `vrf`: `segment["vrf"].lower() == vrf.lower()` (case-insensitive)
  - `vlan_id`: exact int match
  - `allocated=True`: `bool(segment.get("cluster_name")) and not segment.get("released", False)`
  - `allocated=False`: `not bool(segment.get("cluster_name"))`
  - `cluster_name`: exact string match
  - `released`: exact bool match
- Return filtered list.

```python
async def get_segment_by_id(segment_id: str) -> Optional[Dict[str, Any]]:
```
- Call `get_segments()` (unfiltered), return first segment where `segment["_id"] == segment_id`.
